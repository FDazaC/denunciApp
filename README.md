# DenunciApp
DenunciApp pretende ser una aplicación web que facilite la comunicación entre la universidad y los usuarios (funcionarios y estudiantes), para cooperar en el cuidado del entorno del establecimiento. Esto a través de una plataforma de reportes en la que los usuarios puedan registrarse y subir sus observaciones de espacios públicos que esten en mal estado o necesiten mantenimiento.
